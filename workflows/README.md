# TranscriptML Workflows

This directory contains editable Python-cell workflows for interactive use in
VS Code, Spyder, or another editor that understands `# %%` cells.

The checked-in `scripts/` and `scripts/mpra/` directories remain the stable
Sherlock-oriented interfaces. These workflow files are lighter examples for
local exploration and plotting.

Current examples:

- `saluki_visualize_ism_workflow.py`: visualize one Saluki single-nucleotide ISM result with optional metadata lookup and isoform track.
- `mpra_visualize_ism_workflow.py`: visualize one MPRA single-nucleotide ISM result with optional sequence-logo track.
